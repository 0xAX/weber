Welcome to the Weber page.

Weber - is an open source MVC Web framework for Elixir language.

It's also:

  • Server–side templates with EEx
  • JSON generation
  • Language–integrated query syntax
  • and more...
Weber now is very early stage of development, therefore feel free to contribute, Let's make Web with Elixir.
