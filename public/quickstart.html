Quick start

  • Clone https://github.com/0xAX/weber.git repository
  • Execute mix deps.get && mix compile in the weber directory.
  • Execute mix weber /home/user/testWeberApplication
  • Go to the /home/user/testWeberApplication
  • Execute mix deps.get && mix compile
  • ./start.sh and open browser at http://localhost:8080
