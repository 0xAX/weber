Sessions

Weber's session manager automatically started with any web application. Weber has simple API for managing sessions:

  • get_session() -> [] | [{atom, binary()}]
  • get_session(key :: atom()) -> [] | binary()
  • set_session_val(key :: atom(), val :: binary()) -> :ok.
